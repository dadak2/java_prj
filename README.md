# 🚀 Cursor Project

**Cursor 활용 Spring Boot 기반의 종합 웹 서비스 플랫폼**

## 📋 프로젝트 개요

Cursor Project는 Spring Boot 3.5.4를 기반으로 한 현대적인 웹 애플리케이션으로, 다음과 같은 핵심 기능들을 제공합니다:

- **🔐 사용자 인증 시스템**: Spring Security 기반 보안 인증
- **📝 게시판 시스템**: CRUD 기능과 실시간 업데이트
- **🎮 실시간 게임**: 스네이크 게임과 실시간 랭킹 시스템
- **📰 뉴스 피드**: 개인화된 뉴스 피드 서비스
- **⚡ 이벤트 기반 아키텍처**: Redis + Kafka + WebSocket을 활용한 실시간 처리

## 🛠 기술 스택

### Backend
- **Java 17**
- **Spring Boot 3.5.4-SNAPSHOT**
- **Spring Data JPA**
- **Spring Security**
- **Spring Validation**
- **Spring WebSocket**
- **Spring Kafka**
- **Spring Data Redis**
- **Lombok**

### Database
- **H2 Database** (개발용)
- **MariaDB** (운영용)
- **Redis** (실시간 랭킹)

### Message Queue
- **Apache Kafka** (비동기 점수 처리)

### Frontend
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**
- **WebSocket (SockJS + STOMP)**

### Build Tool
- **Gradle**

## 🚀 주요 기능

### 1. 🔐 사용자 관리 시스템
- **모달 기반 인증**: 로그인/회원가입을 모달로 처리
- **JWT 토큰 기반 인증**: Spring Security + JWT
- **사용자 프로필 관리**: 개인정보 및 활동 통계
- **세션 관리**: 자동 로그인 및 세션 유지

### 2. 📝 게시판 시스템
- **실시간 게시판**: 동적 게시글 로딩 및 업데이트
- **고급 검색 기능**: 제목, 내용, 작성자별 검색
- **카테고리 필터링**: 게시글 카테고리별 분류
- **페이지네이션**: 효율적인 대용량 데이터 처리
- **댓글 시스템**: 게시글별 댓글 관리

### 3. 🎮 실시간 게임 시스템
- **스네이크 게임**: HTML5 Canvas 기반 웹 게임
- **실시간 점수 수집**: 게임 종료 시 자동 점수 제출
- **Redis 기반 랭킹**: ZSet을 활용한 고성능 랭킹 시스템
- **WebSocket 실시간 업데이트**: 모든 클라이언트 동기화
- **연결 상태 모니터링**: 실시간 연결 상태 표시

### 4. 📰 뉴스 피드 시스템
- **개인화된 피드**: 사용자 관심사 기반 뉴스 추천
- **카테고리별 분류**: 다양한 뉴스 카테고리 지원
- **팔로우 시스템**: 관심 있는 주제 팔로우
- **실시간 업데이트**: 새로운 뉴스 즉시 반영

### 5. ⚡ 이벤트 기반 아키텍처
- **Redis 캐싱**: 고성능 데이터 캐싱
- **Kafka 메시지 큐**: 비동기 이벤트 처리
- **WebSocket 통신**: 실시간 양방향 통신
- **마이크로서비스 준비**: 확장 가능한 아키텍처

### 6. 🛡️ 보안 및 성능
- **BCrypt 비밀번호 암호화**: 안전한 비밀번호 저장
- **CORS 설정**: 크로스 오리진 요청 처리
- **반응형 디자인**: 모바일/데스크톱 최적화
- **성능 모니터링**: 실시간 성능 지표 추적

## 🚀 실행 가이드

### 1. 필수 요구사항
- **Java 17** 이상
- **Redis** (포트 6379) - ✅ 설치 완료
- **Apache Kafka** (포트 9092) - ⏳ 설치 예정
- **MariaDB** (포트 3306) 또는 **H2 Database** - ✅ 설정 완료

### 2. Redis 설치 및 실행 (✅ 완료)
```bash
# Windows (Chocolatey 사용)
choco install redis-64
redis-server

# 또는 Docker 사용
docker run -d -p 6379:6379 redis:latest
```

### 3. Kafka 설치 및 실행 (⏳ 진행 예정)
```bash
# Kafka 다운로드 후 실행
bin/zookeeper-server-start.sh config/zookeeper.properties
bin/kafka-server-start.sh config/server.properties

# 토픽 생성
bin/kafka-topics.sh --create --topic game-scores --bootstrap-server localhost:9092
bin/kafka-topics.sh --create --topic news-created --bootstrap-server localhost:9092
bin/kafka-topics.sh --create --topic user-follow --bootstrap-server localhost:9092
```

### 4. 애플리케이션 실행
```bash
# 프로젝트 빌드
./gradlew clean build

# 애플리케이션 실행
./gradlew bootRun
```
